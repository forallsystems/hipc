Simplifies how organizations publicize learning events and how learners find events.

== Description ==

Open Learning Events was created for the Hive Interoperability & Portability Coalition (HIPC) II: Open Sharing of Learning Opportunities.
See the wiki here: [https://github.com/forallsystems/hipc/wiki](https://github.com/forallsystems/hipc/wiki)
All development for this plugin is available on GitHub [https://github.com/forallsystems/hipc](https://github.com/forallsystems/hipc)

**Open Learning Events utilizes:**
* Custom post types to store and display events
* Dynamically populated venues from your saved venues
* Custom fields that can be used in RSS feeds
* Custom menus and templates for displaying events
* Event sorting by date

This plugin is optimized for use with [Sela](https://wordpress.org/themes/sela/) By automattic Theme

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/open-learning-events` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Events and Venues will appear under Posts in your Dashboard
4. To ensure Navigation custom menu is enabled go to Appearance -> Menus, selected Primary Menu and save.
5. You're ready to begin to add events and venues
